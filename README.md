# Laravel9BlogApp
## アプリケーションの機能
- 記事投稿機能
- 記事一覧表示機能
- 記事詳細機能
- 記事更新機能
- 記事削除機能
- ユーザー登録機能
- ログイン機能

## 使っている技術・ツール
- 開発環境： __Docker__, __docker-compose__
- フレームワーク： __Laravel9.2.0__
- CSSフレームワーク: __Tailwind CSS__
- データベース：　__MySQL8.0__
- Webサーバー：　__Nginx__
- バージョン管理ツール：　__Git__ / __GitHub__
- その他言語： __JavaScript__, __PHP8.1__, __HTML/CSS__, __Sass(scss記法)__
